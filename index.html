Wi-Fi Cheat-Sheet


Scan:
  iw dev wlan1 scan
  airodump-ng wlan1

Monitor:
  ip link set wlan1 down
  iw dev wlan1 set type monitor
  ip link set wlan1 up

Capture:
  airodump-ng wlan1 -c 6 --bssid XX -w cap

AP:
  hostapd hostapd.conf
  dnsmasq -C dnsmasq.conf
